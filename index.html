Krvatox

Service Utility for Windows

Многофункциональная системная утилита для ОС Windows. Инструмент «всё-в-одном» для анализа дискового пространства, очистки системы, работы с ISO-образами и управления настройками.

Полный набор инструментов

Шесть модулей для полного контроля над дисковым пространством и системой Windows.

Анализатор дискового пространства

Сканирует логические диски для поиска файлов. Фильтрация по типу: дубликаты, фотографии, видео, большие файлы (>100 МБ). Сортировка по размеру, алфавиту и группировка по папкам.

Space Audit

Работа с ISO-образами

Монтирование ISO-файлов для просмотра содержимого. Встроенный инструмент записи ISO на USB-накопитель с автоматизацией через diskpart и bootsect.

ISO Tools

Быстрая очистка

Освобождение места на диске в один клик. Очистка Temp, системного кэша, папки SoftwareDistribution\Download для обновлений Windows и корзины.

Fast Cleanup

Диагностика накопителей

Сбор информации о физических дисках через PowerShell. Таблица с типом накопителя (HDD/SSD), моделью, статусом и оценкой здоровья (Healthy / Warning).

Disk Health

Менеджер автозагрузки

Просмотр и удаление элементов автозагрузки. Чтение данных из реестра Windows — ветки HKCU и HKLM\...\CurrentVersion\Run.

Startup Manager

Локализация и темы

Поддержка русского и английского языков. Две палитры на выбор: тёмная тема с золотыми акцентами (Dark/Gold) и серая тема (Gray/Gold).
